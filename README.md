# 📊 Customer Churn Analysis

## 🔍 Overview
This project analyzes customer churn behavior using a dataset of 50,000 users from a music streaming platform.

The goal was to identify key factors influencing churn and derive actionable insights.

---

## 💡 Key Insights

- ⚡ Inactivity is the strongest predictor of churn  
- 📉 Engagement has minimal impact  
- ⏭️ Skip behavior shows little variation  
- 🎧 Listening time is nearly identical across users

Churn is not sudden — it builds over time.  
Inactivity is the earliest and strongest signal of user drop-off.
